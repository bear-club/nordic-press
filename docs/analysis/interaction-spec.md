# DN.se 交互行为规范

> 分析日期：2026-04-17
> 来源：https://www.dn.se

---

## 交互清单

### INT-STICKY-NAV — 滚动吸顶导航

**触发条件**：用户向下滚动超过顶部工具栏区域（约 50-60px）。

**行为**：
1. `CMP-NAV-TOPBAR`（产品链接栏）滚出视野并隐藏。
2. `CMP-NAV-MAIN` 变为吸顶状态（position: fixed/sticky 固定在视口顶部）。
3. 吸顶状态下 DN Logo 缩小为精简内联版本（更小字号，单行显示）。
4. 主导航链接保持完全可见和可操作。
5. `CMP-NAV-SUB` 保持在吸顶主导航下方可见。
6. Prenumerera（红色 CTA）和 Logga in 按钮在吸顶栏中保持可见。

**向上滚动**：用户滚回顶部时，完整的顶部工具栏重新出现，Logo 恢复原始尺寸。

**状态**：
- **默认**：完整头部 = 顶部工具栏 + 大 Logo + 主导航 + 子导航
- **已滚动**：顶部工具栏隐藏、Logo 缩小、主导航吸顶 + 子导航

### INT-CAROUSEL — 水平轮播导航

**适用组件**：`CMP-CAROUSEL-REVIEWS`、`CMP-CAROUSEL-INSANDARE`、`CMP-CAROUSEL-VIDEO`、`CMP-DN-DIREKT-FEED`

**控制元素**：
- "Föregående"（上一组）按钮 — 左箭头
- "Nästa"（下一组）按钮 — 右箭头

**行为**：
1. 项目水平排列，一次显示部分项目（桌面端通常 3-5 项）。
2. 点击 "Nästa" 轮播滚动到下一组项目（平滑水平滚动/滑动动画）。
3. 点击 "Föregående" 向后滚动。
4. 轮播在起始位置时，"Föregående" 按钮**禁用**（灰化显示）。
5. 轮播在末尾位置时，"Nästa" 按钮推测也会禁用（未滚动至末尾观察到）。
6. 在移动端/平板端，轮播可能支持触摸滑动手势导航。

**视觉提示**：
- 禁用按钮显示为灰化状态，设置 `disabled` 属性。
- 未观察到分页圆点或进度指示器。

**滚动行为**：看起来使用 CSS scroll-snap 或 JS 驱动的平滑滚动。

### INT-PAYWALL — 付费墙内容门控

**触发条件**：未以订阅者身份登录的用户访问付费文章。

**行为**：
1. 文章完整显示分类标签、标题、时间戳和首图。
2. 导语段落（ingress）以粗体衬线字体完整显示。
3. 可能部分显示一到两个正文段落。
4. 文字以 CSS 渐变遮罩（从底部向上的白色半透明渐变）逐渐淡出。
5. 渐隐下方出现居中的 `CMP-PAYWALL` 区块：
   - DN Logo（黑色圆形，白色 "DN."）
   - 文字："Är du redan prenumerant, logga in för att fortsätta läsa."
   - "Logga in" 按钮（描边，黑色边框）
   - 红色背景订阅方案卡片，展示手机上的 DN App
   - 标题："Läs gratis i 3 månader!"
   - 价格："3 månader 0 kr, sedan 85 kr/mån i 3 mån. Därefter 169 kr/mån. Ingen bindningstid."
   - 功能说明："Alla artiklar på DN.se och i DN-appen"
   - Klarna 支付组件（嵌入式 iframe）
   - "Godkänn köp" 按钮
   - 法律条款链接（prenumerationsvillkoren 和 personuppgiftspolicy）
6. 付费墙下方显示一篇免费文章（看起来是"免费样本"文章），可完整阅读。

**状态**：
- **未登录**：付费墙可见，内容截断
- **已登录（非订阅者）**：[需确认] 可能与上述相同
- **订阅者**：完整文章可见，无付费墙

### INT-SUBSCRIPTION-POPUP — 移动端订阅弹窗

**触发条件**：用户在移动设备访问站点，通常在首次访问或间隔一段时间后触发。

**行为**：
1. 出现全屏或接近全屏的遮罩层。
2. 背景变暗/模糊。
3. 弹窗包含订阅方案（"Gratis i 3 månader!"）及价格详情。
4. 右上角 "Stäng dialogruta"（X）关闭按钮。
5. 底部 "Inte just nu" 按钮可关闭。
6. 关闭后恢复正常页面视图。

**备注**：这是一个模态对话框，在关闭前会阻止与页面内容的交互。[需确认 — 具体触发条件和频率]

### INT-LOGIN-TOOLTIP — 登录提示气泡

**触发条件**：页面加载时自动出现（非用户操作触发）。在平板/移动端视口下观察到。

**行为**：
1. 在 "Logga in" 按钮附近出现小型提示气泡/弹出框。
2. 内容："Logga in på DN för en bättre upplevelse!" 标题 + "Skapa gratiskonto och få tillgång till spel, nyhetsbrev och mer." 描述文字。
3. 有 "Stäng"（关闭）按钮。
4. 非模态 — 不阻止页面交互。

**行为**：
1. 切换开关出现在文章元数据区域（署名附近）。
2. 默认状态：关闭（未收藏）。
3. 点击切换收藏状态。

**行为**：
- 分享按钮出现在文章操作栏和底部文章工具区域。
- [需确认] 可能打开分享对话框，提供社交媒体、复制链接、邮件等分享选项。

### INT-TOPIC-FOLLOW — 专题关注切换

**触发条件**：用户点击文章中专题标签的 "Följ" 按钮，或专题页头部的关注按钮。

**行为**：
1. `CMP-ARTICLE-TOPICS` 中每个专题标签旁有 "Följ [专题名称]" 切换开关。
2. 专题页（`TPL-TOPIC`）头部有醒目的红色 "Följ" 按钮。
3. 点击切换关注状态。
4. [需确认] 可能需要登录。关注专题后推测会添加到个性化信息流。

### INT-NAV-SUB-SCROLL — 子导航水平滚动

**触发条件**：子导航项超出视口宽度时（主要在窄视口下）。

**行为**：
1. `CMP-NAV-SUB` 中溢出的项目可通过水平滚动访问。
2. 在移动端，用户可左右滑动查看更多子导航项。
3. 未观察到滚动指示器或箭头。
4. 滚动容器在视口边缘裁切。

---

### INT-NAV-MAIN-SCROLL — 主导航水平滚动（移动端）

**触发条件**：在移动端视口（< 768px）下，主导航项可能溢出。

**行为**：
1. 导航项保持在单行水平排列。
2. 溢出的项目可通过水平滑动/滚动访问。
3. 当前激活栏目链接有红色下划线指示。
4. 末尾的 "Meny" 链接指向 `/meny/`（全站菜单页），而非打开下拉菜单。
5. 在任何断点下都不使用汉堡菜单图标。

### INT-BREAKING-NEWS — 突发新闻条更新

**行为**：
1. `CMP-BREAKING-BAR` 项出现在主内容区域顶部。
2. 项目显示红色 "Just nu:" 前缀 + 标题。
3. 多条突发新闻可垂直堆叠。
4. [需确认] 是否无需刷新页面即可自动更新（实时轮询 vs 静态渲染）。

### INT-DIREKT-TIMELINE — DN Direkt 实时更新

**触发条件**：访问 `/direkt/` 页面。

**行为**：
1. 时间线按时间倒序显示新闻项，最新在前。
2. 每项包含时间戳、标题和可选的 "UPPDATERAD"（已更新）时间戳。
3. 红色圆点标记各项。
4. 每项有 "Dela"（分享）按钮。
5. 点击项目展开显示完整内容或跳转到详情页。
6. [需确认] 时间线是否实时自动更新（WebSocket/轮询）或需要手动刷新。

### INT-VIDEO-PLAY — 视频播放交互

**触发条件**：用户点击视频播放区域的 "Spela"（播放）按钮。

**行为**：
1. 视频缩略图显示，叠加播放按钮和时长标签（如 "0:36"）。
2. 点击播放开始视频回放。
3. 视频播放器包含标准控制元素。
4. 视频标题显示在播放器上方/下方。
5. 视频轮播（`CMP-CAROUSEL-VIDEO`）中每个视频独立可播放。
6. 视频播放器嵌入为 `<region>`，role 为 "Videospelare"。

### INT-SEARCH — 搜索交互

**触发条件**：用户点击导航中的 "Sök" 或访问 `/sok/`。

**行为**：
1. 跳转到专用搜索页（`/sok/`）。
2. 搜索输入框，placeholder："Sök artiklar, ämnen och skribenter"。
3. 搜索图标按钮提交搜索。
4. 过滤控件：
   - 排序下拉框：Nyast（最新，默认值）
   - 日期下拉框：När som helst（任何时间，默认值）
   - 复选框："Sök endast i rubriker"（仅搜索标题）
5. 搜索前：显示 "Vanliga sökningar"（热门搜索）及热门搜索链接。
6. 搜索后：[需确认] 结果以文章卡片形式展示。
7. 侧栏 "DN Arkivet" 卡片建议用户到历史档案深度搜索。

### INT-LAZY-LOAD — 内容懒加载

**触发条件**：用户在首页或长列表页面向下滚动。

**行为**：
1. 图片使用懒加载（在图片 URL 中观察到 `loading="lazy"` 模式）。
2. 图片通过 `static.bonniernews.se` CDN 分发，支持自动格式（webp）、质量、宽度和裁剪参数。
3. 图片 URL 模式：`https://static.bonniernews.se/ba/{uuid}.jpeg?io=1&auto=webp&quality=60&width={w}&crop=16:9,smart`
4. [需确认] 是否有额外内容块在滚动时动态加载（无限滚动），还是页面完全由服务端渲染。

### INT-AD-SLOTS — 广告展示

**触发条件**：页面加载和滚动。

**行为**：
1. 广告位上方标注 "ANNONS" 文字标签。
2. `CMP-AD-BANNER-TOP`：在导航下方、主内容上方渲染，通常为大尺寸格式。
3. `CMP-AD-SIDEBAR`：右侧栏中多个广告位，穿插在内容组件之间。
4. `CMP-AD-INLINE`：主内容列中内容块之间插入的广告单元。
5. `CMP-AD-NATIVE`：赞助内容卡片，样式与编辑内容相似，但有黄色 "ANNONS" 标签和 "Innehåll från [品牌]" 署名。
6. [需确认] 广告是否通过 Header Bidding、Google Ad Manager 或其他广告技术加载。

## 响应式断点总结

根据观察，DN.se 使用以下近似断点：

| 断点 | 行为 |
|------|------|
| >= 1200px | 完整桌面布局：两栏，所有导航项可见，侧栏存在 |
| 768px - 1199px | 压缩桌面：两栏保持，导航项可能需要水平滚动 |
| < 768px | 单列布局：侧栏内容移到下方或隐藏，所有项目全宽显示 |

**关键响应式行为：**
- 导航：所有宽度下均可水平滚动，永不转换为汉堡菜单
- 侧栏：移动端消失，内容移到主信息流下方或移除
- 轮播：可见项减少，支持触摸滑动
- 文章卡片：移动端全宽，垂直堆叠
- 广告位：移动端移除侧栏广告，保留内嵌广告
- 图片：通过 CDN URL 参数实现响应式（提供不同宽度值）
